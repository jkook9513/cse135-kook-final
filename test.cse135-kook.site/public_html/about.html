Our (fake) shop

Juniper & Ash is a fictional furniture shop built to generate analytics events: navigation, clicks, scrolls, typing, and form submissions.

Design principles

  • Minimal dependencies (none).
  • Fast to paste into Apache DocumentRoot.
  • Plenty of interactive UI for logging.
